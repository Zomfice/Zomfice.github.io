Markdown语法

1. 标题

# 一级标题
## 二级标题
### 三级标题

以此类推，总共六级标题，建议在井号后加一个空格，这是最标准的 Markdown 语法。

2. 列表

- 或 * 变为无序列表 

文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格变为有序列表

3. 引用

只需要在文本前加入 > 这种尖括号（大于号）即可

4. 图片与链接

图片为：![]()
链接为：[]()

5. 粗体与斜体

** 粗体 **
* 斜体 *

6. 表格

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
